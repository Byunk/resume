# Resume

LaTeX source for my professional resume.

## Files

- `resume.tex` - LaTeX source file
- `resume.pdf` - Compiled PDF output

## Building

To compile the resume, use a LaTeX distribution (e.g., `pdflatex`):

```bash
pdflatex resume.tex
```

## Requirements

The resume uses the following LaTeX packages:
- `geometry`
- `titlesec`
- `enumitem`
- `hyperref`
- `fontawesome5`
- `xcolor`

Make sure these packages are installed in your LaTeX distribution.
